# 发给朋友玩的方式

当前 `http://127.0.0.1:8765/` 只在你的电脑上可用，`http://192.168.1.103:8765/` 只在同一个 Wi-Fi / 局域网里可用。朋友在外网要玩，需要一个公网地址。

## 大陆长期访问：优先国内云 + 备案

如果目标用户在中国大陆，并且未来要接广告、充值，推荐优先使用：

- 阿里云 / 腾讯云 / 华为云 / 火山引擎等国内云服务器、对象存储或静态托管
- 国内 CDN
- 自有域名
- ICP 备案
- 公安联网备案

GitHub Pages、Cloudflare Pages、Netlify、Vercel 适合海外试玩，不适合作为大陆商业化主站，因为访问稳定性、备案、广告和支付接入都会受限制。

上传内容就是 `outputs/` 目录里的这些文件：

- `index.html`
- `star-dodge.html`
- `assets/`
- `data/`
- `MAINTENANCE.md`

## 最省事的上传包

我已经准备了 `arcade-site.zip`。它适合用于测试上传。正式大陆运营时，建议把解压后的文件放到已备案域名对应的国内云资源上。

## 临时隧道为什么不推荐

临时隧道可以把本地电脑临时映射成公网链接，但它依赖：

- 电脑保持开机
- 本地服务持续运行
- 网络允许隧道连接
- 临时域名不失效

本机当前网络对隧道通道有限制，所以更适合直接部署静态站。

## 长期维护建议

新增游戏时：

1. 把新游戏 HTML 放到站点根目录。
2. 更新 `data/games.json`。
3. 重新上传或触发平台部署。

这样朋友访问的链接不会变，游戏库会持续更新。

## 商业化提醒

- 广告：优先接国内广告联盟或平台广告能力，不要依赖 Google、海外 CDN 或海外统计脚本。
- 充值：不能只靠静态页面，需要后端订单、支付回调、风控、实名/防沉迷、退款和对账。
- 游戏运营：公开运营、内购、上架平台前，需要确认游戏版号、实名、防沉迷、未成年人充值限制等要求。
- 数据：账号、充值、进度、广告归因都涉及个人信息处理，需要隐私政策、用户协议和数据安全措施。
